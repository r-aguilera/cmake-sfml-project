# CMake SFML Project Custom Template

This template is based on the [CMake SFML Template](https://github.com/SFML/cmake-sfml-project) (Public Domain version) and is intended for private projects.  
For information on building or running the original project, refer to its repository linked above.

This template is also released into the public domain and provided "as is."

An example license template is included; rename and customize it for each derived project.
